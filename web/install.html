Install

You need to enable JavaScript on this page to proceed with the Chickenfoot installation.

Click the link below to install Chickenfoot. You will have to restart Firefox after installation for the changes to take effect. (This means that you will have to close ALL of your Firefox windows in order to restart.)

Your browser does not appear to have the Java 5.0 plugin for Firefox. Some features of Chickenfoot, such as packaging a script as a Firefox extension, require the Java 5.0 plugin. Try these instructions for installing the plugin. Most features of Chickenfoot, such as creating and editing scripts, will still be available to you.
